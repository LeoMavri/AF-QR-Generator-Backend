# AF-QR-Code-Generator-Backend

Scris cu:

-   Typescript
-   Typeorm
-   Express.js
-   Zod

## Instalare

-   Instalare Node.js (v20 LTS)
-   Instalare PostgreSQL
-   Creare baza de date cu numele setat in `.env`
-   `npm install`
-   `npm run start`

## Env

```env
PORT=API_PORT_HERE
DATABASE_HOST=
DATABASE_PORT=
DATABASE_NAME=
DATABASE_USERNAME=
DATABASE_PASSWORD=
AUTH_HEADER=UNUSED
```
